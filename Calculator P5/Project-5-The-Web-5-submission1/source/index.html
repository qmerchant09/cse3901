Calculator Tutorial


Entering an Expression
To enter an expression, hit the number buttons(0-9) and operator buttons(+-*/).
You can also use the subtraction operator as a negative.
Once you have finished creating your expression, hit the = button to get your result.
Clearing an Element (CE)
Clears the last element currently displayed on the calculator.
Clearing the Display (C)
Clears the full expression currently displayed/being calculated including the latest element.
Display History/Memory
On the right side of the screen are two areas where your memory and history are stored.
After every expression is entered, it is stored in the history.
To store a value currently on the display into memory hit the MS button.
Adding to Memory (MC, MS, MR, M+, M-)
MC : Clears all entries from memory.
MS : Saves the current number displayed to the screen into memory.
MR : Displays the most recent value in memory
M+ : Adds the current value on screen to the most recent memory entry.
M- : Subtracts the current value on screen from the most recent memory entry
